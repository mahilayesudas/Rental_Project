# Residential Property Portfolio Analytics Dashboard

## Project Overview

This project delivers an end-to-end analytics solution for a residential property management portfolio of 14 properties.

The solution automates financial data extraction from 80 income statement PDFs using Python and integrates municipal tax data to build a Power BI dashboard for performance monitoring.

## Business Problem

Financial data was distributed across multiple PDF income statements with no centralized reporting. The goal was to automate extraction, integrate tax data, and build an interactive dashboard for profitability analysis and expense monitoring.

## Project Architecture

Income Statement PDFs (80)
→ Python Extraction & Cleaning
→ CSV Dataset
→ Power BI Data Model
→ Interactive Dashboard & KPIs

## Data Sources

* 80 Income Statement PDFs
* Municipal Property Tax Data (City of Kingston)
